#String

```
.mq_substring(with range : Range<Int>) -> String
```

获取[st, ed)位置子串, 支持负数操作, 边界判断

由于`swift`的`Range`不支持前大后小, 所以没法像`python`般灵活地切片

测试字符串: "" | 运行结果
:--------:|:--:
-2..<(-1) | ""
-1..<0    | ""
0..<1		| ""

测试字符串: "12345" | 运行结果
:--------:|:--:
-2..<(-1) | "4"
-1..<0    | "5"
0..<1		| "1"

.mq_substring(with range : ClosedRange<Int>) -> String
